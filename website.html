About This Demo

This web application demonstrates webcam access for hand tracking. In a full implementation, this would:

  • Capture live video from your webcam
  • Process each frame using MediaPipe for hand detection
  • Draw hand landmarks and connections on the video feed
  • Track hand movements in real-time

Note: Make sure to allow camera access when prompted by your browser.
